Zustand 스토어 책임 분리
projectStore
  • 등록 프로젝트 목록
  • 활성 프로젝트
  • 프로젝트 유형/아이콘
fileTreeStore
  • 트리 데이터
  • 펼침/접힘 상태
  • 선택 파일
  • 필터 상태
viewerStore
  • 현재 문서 내용
  • 스크롤 위치
  • 탐색 히스토리
  • TOC 항목
uiStore
  • 테마 (dark/black)
  • 패널 너비
  • 사이드바 접힘
